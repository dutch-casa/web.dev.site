# Web Playground

Experiment with code directly in your browser using our interactive playground. The playground runs a full development environment powered by WebContainers.

## Try It Yourself

Edit the HTML below and click Run to see your changes live. Try changing the colors, the text, or add new elements!

## How It Works

The playground runs a complete Node.js environment directly in your browser using WebContainers. When you click Run:

1. **Boot** - The WebContainer initializes
2. **Mount** - Your files are loaded into the virtual filesystem
3. **Install** - Dependencies from package.json are installed
4. **Start** - The dev server spins up and serves your application

## Challenge

Try modifying the code to:

- Change the gradient colors on the body background
- Update the card's border-radius for a different look
- Add a new element to the card
- Change the button's hover effect

<Tip>
The playground supports hot module replacement (HMR), so changes you make will automatically reload in the preview without needing to click Run again.
</Tip>
